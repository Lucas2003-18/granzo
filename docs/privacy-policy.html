Sobre o Granzo

O Granzo é um aplicativo de finanças pessoais para Android desenvolvido para uso individual. Ele permite registrar lançamentos financeiros, importar extratos bancários, acompanhar orçamentos e conversar com uma IA para análise das finanças.

🗄️

Dados que coletamos

O Granzo não coleta, não transmite e não armazena dados pessoais em servidores externos. Todas as informações ficam exclusivamente no seu dispositivo:

  • Lançamentos financeiros que você cadastra manualmente
  • Extratos importados via arquivo CSV
  • Configurações de categorias, orçamentos e metas
  • Chave de API do Gemini (armazenada localmente)
🔒

Seus dados financeiros nunca saem do seu celular. Não há cadastro, login ou sincronização com servidores.

🤖

Integração com IA (Gemini)

O recurso de IA utiliza a API do Google Gemini. Ao usar o chat de IA, um resumo anonimizado das suas finanças (totais por categoria, sem descrições individuais) é enviado à API do Google para gerar análises. Isso ocorre apenas quando você inicia uma conversa na aba IA.

Consulte a Política de Privacidade do Google para entender como os dados enviados à API Gemini são tratados.

📤

Compartilhamento de dados

O Granzo não compartilha, não vende e não monetiza seus dados com terceiros. Não há publicidade, rastreamento ou analytics integrados ao aplicativo.

🗑️

Exclusão de dados

Para excluir todos os seus dados, acesse Config → Dados → Apagar todos os dados dentro do aplicativo. Todos os lançamentos, configurações e histórico serão removidos permanentemente do dispositivo.

👶

Uso por menores

O Granzo não é direcionado a menores de 13 anos. O aplicativo não coleta intencionalmente dados de crianças.
